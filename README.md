install solc
> sudo add-apt-repository ppa:ethereum/ethereum

> sudo apt-get update

> sudo apt-get install solc


npm install

1. `npm install -g ethereumjs-testrpc http-server`
2. `cd simplecoin`
3. `npm install`

How to run

1. `npm run testrpc`
2. `node deploy.js`
3. `http-server`
4. checkout http://127.0.0.1:8000

Files

* app.js
  前端代码

* coin.sol
  智能合约

* deploy.js
  部署合约的脚本

* index.html
  页面
